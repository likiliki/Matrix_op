Operaciones con Matrices

 

En esta Ventana se realizan operaciones con dos matrices Previamente almacenadas, las operaciones disponibles son Suma, Resta y Producto.

Las matrices resultantes son guardadas por la aplicación para ver los resultados mas tarde o para trabajar  con ella en una nueva operación. Para eso debe pasarla a matrices operación, en el menú de Administración de matrices.

 

Operaciones Matriz - Numero

 

En esta Ventana se realizan operaciones de una matriz con un numero, se realizan operaciones de suma resta producto y división de matriz y numero. Estas operaciones no son guardadas.

 

También se realizan operaciones especiales con una sola matriz, como la inversa o el método de Gauss, estas operaciones especiales si son guardadas por la aplicación.

 

Administración de matrices

 

Las matrices almacenadas se dividen en Matrices Operar y Resultado, solo se permiten las matrices Operar para realizar operaciones.

En esta Ventana se permite modificar y eliminar matrices y pasar las matrices resultado a matriz principal para poder operar con las matrices resultado.

 

Mostrar Datos HTML

 

Ventana que muestra un archivo html en el que son guardados los datos de manera automática.   Las matrices estan divididas según el grupo al que pertenezcan.
